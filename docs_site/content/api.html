基础地址和请求规范

本地 API：http://localhost:50039。所有业务接口以 /v1 开头；健康检查是 /health。接口返回 JSON，异步提交通常返回 HTTP 202。

i

不要让调用方直接依赖数据库表。Web API 是 Dataset、Scene、质量、入库和发布状态的业务边界。

认证和公开入口

认证开启时，除 POST /v1/partition/schemas/import 外，/v1/* 默认要求 Bearer Token：

curl -H 'Authorization: Bearer <your-token>' \
  http://localhost:50039/v1/partition/load-batches

前端非管理员只显示公共编码入口，直接访问剖分页会回门户首页；这不是后端授权的替代品。

常见 HTTP 状态

状态码含义处理方式
401缺少或无效 token重新获取身份 token
403角色没有权限联系管理员，不要关闭认证
404业务对象不存在核对 ID 是否来自最新查询
409状态机或并发冲突刷新详情，按当前状态操作
422请求契约不合法看 detail，修正字段后重试
503后端依赖未配置或不可用检查 OpenGauss、Ray、MinIO 和服务配置

格网 SDK 入口

Web 的 locate、cover、topology、code 接口调用进程内 CubeEncoderSDK，不需要另起 encoder HTTP 服务。其他包不得复制格网逻辑；调用方只使用 geohash、mgrs、isea4h。
